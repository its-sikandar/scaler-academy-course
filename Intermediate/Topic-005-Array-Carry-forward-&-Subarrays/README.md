## Key Concepts
- **Carry Forward Technique:** Optimize by carrying information from previous iterations
- **Subarrays:** Contiguous parts of an array (Total count: N(N+1)/2)
- **Position Tracking:** Track last seen positions for optimization
- **Time Complexity:** Carry forward reduces O(N²) to O(N)
