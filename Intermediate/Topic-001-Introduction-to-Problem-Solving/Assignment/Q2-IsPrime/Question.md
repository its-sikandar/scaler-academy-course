#  IsPrime

## Prime Number: Positive Number with exactly 2 factors, ie 1 and itself

###Problem Description

```
Given a number A. Return 1 if A is prime and return 0 if not. 

Note : 
The value of A can cross the range of Integer.

```



###Problem Constraints

```
1 <= A <= 10^9
```


### Example Explanation

## Explanation 1:
```5 is a prime number.```

Explanation 2:

```10 is not a prime number..```
