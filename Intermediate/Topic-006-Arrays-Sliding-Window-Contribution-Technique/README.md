## Key Concepts
- **Sliding Window:** Fixed or variable-size window that slides through array - O(N)
- **Contribution Technique:** Calculate how many times each element appears in all subarrays
- **Formula:** Element at index i appears in (i+1) × (N-i) subarrays
- **Kadane's Algorithm:** Maximum subarray sum in O(N) time
- **Applications:** Range queries, subarray sums, pattern matching
